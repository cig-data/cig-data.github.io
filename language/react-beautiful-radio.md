## 怎么使用

### 安装

```bash
$ npm i react-beautiful-radio -S
```

## 组件基本参数说明

以下描述 @cig/radio-block 里可使用的基本参数。
  

## API 介绍

| Property | Description | Type | Default |
| --- | --- | --- | --- |
| checked | 是否选中 | bool | false |
| disabled | 是否禁用 | bool | false |
| value | 描述文本 | string | - |
| onClick | 点击事件 | function(e){} | - |


> 以上为 @cig/radio-block 里可支持的参数
